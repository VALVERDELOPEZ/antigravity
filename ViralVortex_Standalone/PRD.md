# Viral Vortex PRD: The Ultimate Viral Engine

## 1. Vision
Viral Vortex is an autonomous system that creates "perfectly imperfect" viral content by blending psychological triggers with human-mimicry automation.

## 2. Core Features (Data-Driven)
### A. The STEPPS Evaluator (Science-Based)
- Analyzes content against Jonah Berger's 6 pulses (Social Currency, Triggers, Emotion, Public, Practical Value, Stories).
- Only allows distribution if "Virality Score" > 8.5.

### B. "Humanizer" Layer (User-Need Based)
- Post-processing content to remove "AI signatures".
- Injection of regional slang, intentional minor typos (character-swaps), and variadic formatting.

### C. Vibe-Check Sentinel (Market-Based)
- Real-time analysis of target communities (Subreddits, Twitter Lists).
- Detects "toxic" or "mod-heavy" periods to pause automation.

### D. Multi-Persona Mesh (Memory-Based)
- Database of persistent personas with consistent backstories.
- Prevents cross-token/IP correlation.

## 3. Tech Stack
- **Engine**: Python (Agentic Logic)
- **Orchestration**: n8n (Distribution)
- **Database**: Supabase (Persona & Sentiment memory)
- **AI**: Claude 3.5 Sonnet (Refinement) + GPT-4o (Evaluating)

## 4. Success Metrics
- Average Engagement Rate (ER) > 5% per post.
- Account Longevity > 90 days.
- Virality Conversion (Post -> Lead) tracking.
